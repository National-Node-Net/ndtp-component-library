# README  

**Repository:** `ndtp-component-library`  
**Description:** `Front-end Component Library solution for the National Digital Twin ecosystem`  
**SPDX-License-Identifier:** `Apache-2.0 AND OGL-UK-3.0 `  

## Overview  

This repository has been put together to store front end components that developers working on other repositories within the program can use to ensure a consistent look and feel throughout all user interfaces within the ecosystem.  

This repository will be developed over time, populating the repository with reusable components alongside the development of other projects (e.g. Management Node UI, Data Catalogue)

## Prerequisites  

There are currently no prerequisites required to make use of this repository

## Quick Start  

There are currently no quickstart steps required to run the component within this application. Simply copy and paste the relevant code for components you wish to us into your own repository.

## Features  

- **Reusable components to be used in the development of user interfaces in other repositories**
